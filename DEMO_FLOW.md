## ⚠️ COMMON ISSUES & FIXES

**Issue:** Dashboard shows empty incident queue  
**Fix:** Click "Simulate Attack" button to ingest data

**Issue:** Honeypot doesn't start  
**Fix:** Make sure port 8888 is not in use: `netstat -ano | findstr :8888`

**Issue:** Gemini responses missing  
**Fix:** Check `.env` file in backend has `GEMINI_API_KEY=your_key_here`

**Issue:** Frontend won't load  
**Fix:** Make sure backend is running on port 8000 first

**Issue:** "Connection refused" on API calls  
**Fix:** Backend crashed. Check terminal for errors. Restart: `python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000`
